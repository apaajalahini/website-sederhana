Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif. JavaScript dapat digunakan untuk mengubah konten halaman secara dinamis, menangani event, dan berinteraksi dengan server tanpa harus memuat ulang halaman.

Struktur Dasar JavaScript

JavaScript biasanya diletakkan dalam tag <script>. Berikut adalah contoh struktur dasar JavaScript:

<script>
    console.log("Halo, Dunia!"); // Ini adalah komentar
</script>

Contoh di atas akan menampilkan pesan "Halo, Dunia!" di konsol pengembang (developer console) browser.

Variabel dan Tipe Data

JavaScript mendukung berbagai tipe data seperti string, number, boolean, dan objek. Variabel di JavaScript dapat dideklarasikan menggunakan kata kunci let, const, atau var. Berikut adalah contoh penggunaan variabel di JavaScript:

<script>
    let nama = "John";
    let umur = 25;
    console.log("Nama saya " + nama + " dan saya berumur " + umur + " tahun.");
</script>

Contoh di atas akan menampilkan "Nama saya John dan saya berumur 25 tahun." di konsol.

Struktur Kontrol

JavaScript memiliki berbagai struktur kontrol seperti if, else, for, dan while. Berikut adalah contoh penggunaan if untuk memeriksa kondisi:

<script>
    let x = 10;
    if (x > 5) {
        console.log("X lebih besar dari 5");
    } else {
        console.log("X kurang dari atau sama dengan 5");
    }
</script>

Contoh di atas akan menampilkan "X lebih besar dari 5" di konsol.

Fungsi dalam JavaScript

JavaScript memungkinkan Anda untuk membuat fungsi yang dapat digunakan berulang kali. Berikut adalah contoh fungsi untuk menambahkan dua angka:

<script>
    function tambah(a, b) {
        return a + b;
    }

    console.log(tambah(5, 3)); // Output: 8
</script>

Contoh di atas mendefinisikan fungsi tambah yang mengembalikan hasil penjumlahan dua angka.

Penggunaan Event dalam JavaScript

JavaScript memungkinkan Anda untuk menangani event seperti klik tombol atau pengisian formulir. Berikut adalah contoh menggunakan event click pada sebuah tombol:

<script>
    function klikTombol() {
        alert("Tombol diklik!");
    }
</script>


<button onclick="klikTombol()">Klik Saya

Contoh di atas akan menampilkan sebuah alert saat tombol diklik.

Menangani Input Pengguna

JavaScript juga dapat digunakan untuk menangani input pengguna. Berikut adalah contoh menggunakan JavaScript untuk membaca input dari formulir HTML:

<script>
    function tampilkanData() {
        let nama = document.getElementById("nama").value;
        alert("Nama Anda: " + nama);
    }
</script>


<input type="text" id="nama" placeholder="Masukkan Nama">
<button onclick="tampilkanData()">Kirim

Formulir ini memungkinkan pengguna untuk memasukkan nama, dan saat tombol diklik, JavaScript akan menampilkan nama yang dimasukkan melalui alert.
