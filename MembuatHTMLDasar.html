Dasar-dasar HTML untuk Pemula

Apa itu HTML?

HTML (Hypertext Markup Language) adalah bahasa standar yang digunakan untuk membuat halaman web. HTML menyusun struktur dasar dari sebuah halaman web.

Struktur Dasar HTML

Berikut adalah struktur dasar dari sebuah halaman HTML:

<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo, Dunia!</h1>
    <p>Selamat datang di tutorial HTML!</p>
  </body>
</html>

Penjelasan:

  • <!DOCTYPE html> - Menentukan jenis dokumen dan versi HTML (HTML5 di sini).
  • <html> - Elemen utama yang mengelilingi seluruh dokumen HTML.
  • <head> - Bagian yang berisi metadata seperti karakter set dan judul halaman.
  • <body> - Bagian yang berisi konten yang ditampilkan pada halaman web.

Tag HTML Dasar

HTML menggunakan tag untuk menentukan elemen di halaman web. Beberapa tag dasar yang sering digunakan adalah:

  • <h1> sampai <h6> - Tag untuk judul, dengan <h1> menjadi yang terbesar dan <h6> yang terkecil.
  • <p> - Tag untuk paragraf.
  • <a> - Tag untuk membuat tautan (hyperlink).
  • <img> - Tag untuk menampilkan gambar.
  • <ul> - Tag untuk daftar tidak terurut (bullet list).
  • <ol> - Tag untuk daftar terurut (numbered list).

Contoh Penggunaan Tag

Tag Judul (Headings)

<h1>Ini adalah judul level 1</h1>
<h2>Ini adalah judul level 2</h2>
<h3>Ini adalah judul level 3</h3>

Tag Paragraf

<p>Ini adalah paragraf pertama.</p>
<p>Ini adalah paragraf kedua.</p>

Formulir HTML

HTML memungkinkan pembuatan formulir dengan tag <form>. Berikut adalah contoh formulir sederhana:

<form>
  <label for="nama">Nama</label>
  <input type="text" id="nama" name="nama"><br>
  
  <label for="email">Email</label>
  <input type="email" id="email" name="email"><br>
  
  <input type="submit" value="Kirim">
</form>
