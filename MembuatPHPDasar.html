Dasar-dasar PHP untuk Pemula - Panduan Lengkap

Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman yang digunakan untuk pengembangan aplikasi web dinamis. PHP sering digunakan untuk membuat halaman web interaktif dan dapat berinteraksi dengan database.

Struktur Dasar PHP

PHP dimulai dengan tag pembuka <?php dan diakhiri dengan tag penutup ?>. Berikut adalah contoh struktur dasar kode PHP:

<?php
    echo "Halo, Dunia!"; // Ini adalah komentar
?>

Contoh di atas akan menampilkan teks "Halo, Dunia!" di halaman web.

Variabel dan Tipe Data

PHP mendukung berbagai tipe data seperti string, integer, float, dan boolean. Variabel di PHP diawali dengan simbol $. Berikut adalah contoh penggunaan variabel di PHP:

<?php
    $nama = "John";
    $umur = 25;
    echo "Nama saya $nama dan saya berumur $umur tahun.";
?>

Contoh di atas akan menampilkan teks "Nama saya John dan saya berumur 25 tahun."

Struktur Kontrol

PHP memiliki berbagai struktur kontrol seperti if, else, dan while. Berikut adalah contoh penggunaan if untuk memeriksa kondisi:

<?php
    $x = 10;
    if ($x > 5) {
        echo "X lebih besar dari 5";
    } else {
        echo "X kurang dari atau sama dengan 5";
    }
?>

Contoh di atas akan menampilkan "X lebih besar dari 5".

Fungsi dalam PHP

PHP memungkinkan Anda untuk membuat fungsi yang dapat digunakan berulang kali. Berikut adalah contoh fungsi untuk menambahkan dua angka:

<?php
    function tambah($a, $b) {
        return $a + $b;
    }

    echo tambah(5, 3); // Output: 8
?>

Contoh di atas mendefinisikan fungsi tambah yang mengembalikan hasil penjumlahan dua angka.

Penggunaan Formulir dengan PHP

PHP dapat digunakan untuk memproses data yang dikirim melalui formulir HTML. Berikut adalah contoh formulir HTML yang mengirim data ke PHP untuk diproses:


<form action="proses.php" method="post">
    Nama: <input type="text" name="nama"><br>
    Umur: <input type="text" name="umur"><br>
    <input type="submit" value="Kirim">
</form>


<?php
    $nama = $_POST['nama'];
    $umur = $_POST['umur'];
    echo "Nama: $nama, Umur: $umur";
?>

Formulir ini mengirim data ke file PHP untuk diproses dan kemudian menampilkan data yang diterima.

Menampilkan Data dari Database MySQL

PHP sering digunakan untuk berinteraksi dengan database MySQL. Berikut adalah contoh kode PHP untuk menampilkan data dari database:

<?php
    // Menghubungkan ke database
    $conn = new mysqli("localhost", "username", "password", "database");

    // Memeriksa koneksi
    if ($conn->connect_error) {
        die("Koneksi gagal: " . $conn->connect_error);
    }

    // Query untuk mengambil data
    $sql = "SELECT id, nama FROM users";
    $result = $conn->query($sql);

    if ($result->num_rows > 0) {
        // Menampilkan data
        while($row = $result->fetch_assoc()) {
            echo "ID: " . $row["id"]. " - Nama: " . $row["nama"]. "
";
        }
    } else {
        echo "0 hasil";
    }
    $conn->close();
?>

Contoh di atas menampilkan data dari tabel "users" di database MySQL.
