Dasar-dasar CSS untuk Pemula

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web. CSS digunakan untuk mengatur warna, ukuran, posisi elemen, dan banyak lagi.

Struktur Dasar CSS

CSS terdiri dari selector, properti, dan nilai. Berikut adalah struktur dasar CSS:

/* Ini adalah komentar */
selector {
    property: value;
}

Contoh:

h1 {
    color: #3498db;
    font-size: 36px;
}

Tag HTML yang Dapat Diberi Gaya

Beberapa tag HTML yang sering diberi gaya menggunakan CSS adalah:

  • <h1> sampai <h6> - Tag untuk membuat judul (heading).
  • <p> - Tag untuk membuat paragraf.
  • <a> - Tag untuk membuat tautan (link).
  • <div> - Tag untuk membuat divisi atau kontainer.
  • <img> - Tag untuk menampilkan gambar.

Menambahkan Warna pada Teks

CSS dapat digunakan untuk mengubah warna teks. Berikut adalah contoh CSS untuk mengubah warna teks pada tag <h1>:

h1 {
    color: #3498db; /* Mengubah warna teks menjadi biru */
}

Menambahkan Padding dan Margin

Padding adalah jarak antara konten dan batas elemen, sementara margin adalah jarak antara elemen dengan elemen lainnya. Berikut adalah contoh CSS untuk padding dan margin:

p {
    padding: 10px;  /* Menambahkan padding 10px di dalam elemen */
    margin: 20px;   /* Menambahkan margin 20px di luar elemen */
}

Menambahkan Border

CSS juga memungkinkan Anda menambahkan border (garis) pada elemen HTML:

div {
    border: 2px solid #3498db; /* Menambahkan border biru dengan ketebalan 2px */
    padding: 15px;
}

Menambahkan Background Color

CSS memungkinkan Anda untuk memberi warna latar belakang pada elemen:

section {
    background-color: #f0f4f8; /* Menambahkan warna latar belakang */
}

CSS Box Model

Box model adalah konsep yang menjelaskan bagaimana elemen HTML dirender di browser. Setiap elemen memiliki margin, border, padding, dan konten. Berikut adalah gambaran umum box model:

  • Margin: Ruang di luar border.
  • Border: Garis yang mengelilingi padding dan konten.
  • Padding: Ruang antara border dan konten.
  • Konten: Isi dari elemen (misalnya teks atau gambar).

Contoh penggunaan box model:

div {
    margin: 20px;
    border: 2px solid #3498db;
    padding: 15px;
}
